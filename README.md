Featured Project
Astakos AI Agent
Astakos is a personal AI operating layer, not just a chatbot wrapper.
It is designed to:
remember important facts, events, and preferences
learn recurring routines from daily life
send proactive reminders with context
reason across chat, files, photos, voice, and documents
stay local-first, with memory and runtime state stored on the user's machine
Core stack: Python LangGraph FastAPI SQLite ChromaDB Gemini / Vertex AI
